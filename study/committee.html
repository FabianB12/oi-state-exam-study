Common 1 · Polynomial graph algorithms, search trees, automata

BE4M33PALlikely
  • Notation of asymptotic complexity. Graph vocabulary: degree, path, circuit, cycle. Graph representations by adjacency, distance, Laplacian and incidence matrices. Adjacency lists.
  • Minimum spanning tree: Prim-Jarník, Kruskal, Borůvka. Strongly connected components: Kosaraju-Sharir, Tarjan. Euler trail. Union-find. Graph and tree isomorphism.
  • Generation of combinatorial objects: subsets, k-element subsets, permutations. Gray codes. Primes, sieve of Eratosthenes. Pseudorandom numbers, linear congruential generator.
  • Search trees: binary, AVL, red-black, B-tree and B+ tree, splay tree, k-d tree and nearest-neighbor searching in k-d trees. Skip list.
  • Finite automata, regular expressions, text search: exact and approximate matching (Hamming/Levenshtein), dictionary automata.

Common 2 · Complexity classes, Turing machines, undecidability

BE4M01TALmaintain only
  • Asymptotic growth, time/space complexity, correctness via variant and invariant.
  • Deterministic, multitape, and nondeterministic Turing machines.
  • Classes P, NP, co-NP. Polynomial reduction, NPC, Cook theorem. Heuristics and approximation for NP-complete problems.
  • PSPACE and NPSPACE, Savitch theorem. Randomized classes RP, ZPP, co-RP.
  • Decidability: recursive and recursively enumerable languages, diagonal language, universal language.

Common 3 · Combinatorial optimization

BE4M35KOlikely
  • Integer linear programming: shortest-path and TSP ILP formulations, Branch and Bound, polynomially solvable ILP classes.
  • Shortest paths: Dijkstra, Bellman-Ford, Floyd-Warshall, DAGs, problem formulations.
  • Network flows: max flow/min cut, Ford-Fulkerson, feasible flow with balances, min-cost flow and cycle canceling, matching.
  • Knapsack: approximation, dynamic programming, approximation scheme.
  • TSP: double-tree and Christofides for the metric case, local search k-OPT.
  • Scheduling: notation, Bratley, Horn, list scheduling, project scheduling with temporal constraints.
  • Constraint Satisfaction Problem. AC3 algorithm.
